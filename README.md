## 📌 프로젝트 설명
**구독 매니아**는 사용자가 직접 구독 중인 서비스를 등록하고,  
결제일 알림을 이메일로 받아볼 수 있는 **개인 구독 관리 서비스**입니다.

- 카테고리별로 다양한 구독 플랫폼을 조회할 수 있습니다.
- 각 플랫폼에 대해 사용자가 리뷰(피드백)를 남기고 평가할 수 있습니다.
